Time to improvse the Model¶

There are 2 improvements 1) Adding priorities to Expirence replays..
2) Adding more layers to neural network

In Expirence reply technique we used equal random policies.. No weights(probabilities) given to specific frame. But in this technique we use Prioritized expirence reply. Which mean we add probabilitied to each frame(state)..

We add more weight to frame we got more error. So it get trained better

But this needs bias correction. so we divide the number of samples x probablities raised to power of beta Also while adding probabilities we each state we will normalize it via (probabiliy /sum(probabluities)^ alpha We also need to add minimum probabilities to state so that it wonot be ignored

Adding more neural network layers reduces more bias
